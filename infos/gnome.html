Gnome Install

Before to follow this instruction, please, check if you have this or similar make.conf in your /etc directory.
  1. These two ports should be installed before everything:

    • math/djbfft
    • audio/lame

  2. For to install gnome these ports should be installed:

    • x11/gnome2
    • x11/gnome2-fifth-toe
    • x11/gnome2-power-tools
    • devel/gnome2-hacker-tools
    • editors/gnome2-office
    • www/linuxpluginwrapper
    • www/mplayer-plugin
    • x11-fonts/webfonts
    • x11/xorg
    • print/gnome-cups-manager
    • print/cups
    • java/jdk14

References:

  • "[In Russian] GNOME 2.6.0 setup on FreeBSD" by Sergey Akifyev.
